# Midterms E-commerce Frontend

## Project Description

This project is a single-page e-commerce frontend application built with React. It provides a shopping flow with dedicated pages for home, product listing, product details, and cart management. Navigation is handled through client-side routing to create a smooth user experience across pages.

## Tech Stack

- React 19
- React DOM 19
- React Router DOM 6
- Vite 6
- ESLint 9
- JavaScript (ES Modules)
- CSS
